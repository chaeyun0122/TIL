# Kaggle 소개

## What is Kaggle?
> "We're making data science into a sport."
* Dataset : 데이터셋을 공유하고 접근
* Kernel : 데이터셋을 다루는 여러가지 방법들을 만들고 공유
* Cha글llenge : 여러사람들이 어려운 문제에 대해서 도전
* Competition : 데이터로부터 머신러닝 모델을 만들어서 사람들과 성능을 가지고 승부를 겨룸
